**TDP RAM UVC**

**RAM Specification:**

  - Memory Size       : 1 KBytes
  - Address Range     : 0x000 to 0x3FF
  - Address Bus Width : 10
  - Data Bus Width    : 8
  - Byte Size Memory
  - Dual Port Memory
  - Two separate ports
  - Simultaneously Write and Read all combinations possible

**Signals of TDP RAM DUT:**

Input Signals of DUT:

  - we_a
  - re_a
  - addr_a
  - data_a
  - we_b
  - re_b
  - addr_b
  - data_b

Output Signals of DUT:

  - q_a
  - q_b
